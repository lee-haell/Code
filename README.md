# TWIZY

### 주요 기능
1. 반응형
2. 로딩화면 transition 효과
3. 네비게이션 click 이벤트로 if문 활용
4. 메인 비주얼은 세로 슬라이드 구현 (li의 height 만큼 ul이 이동하는 방식)
5. scroll 이벤트에 반응하는 컨텐츠 영역
- HTML5, CSS3, jQuery

- 르노 트위지 웹 사이트 입니다.
